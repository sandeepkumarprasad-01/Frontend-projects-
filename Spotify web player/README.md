# Spotify Web Player Clone

A full-stack Spotify web player clone built with Express.js and Vanilla JavaScript & CSS. This project connects to the official Spotify API to fetch and display music data, featuring a dynamic UI similar to the real Spotify application.

## 🚀 Getting Started Locally

To run this project on your own machine:

1. Clone the repository.
2. Navigate to the `Spotify web player` directory.
3. Run `npm install` to install the backend dependencies.
4. Create a `.env` file in the root of the project and add your Spotify API credentials:
   ```env
   SPOTIFY_CLIENT_ID=your_client_id
   SPOTIFY_CLIENT_SECRET=your_client_secret
   ```
5. Run `npm start` to start the Express server.
6. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Node.js, Express.js, Axios
